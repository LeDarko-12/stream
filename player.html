💡 Guía de integración del reproductor

Para integrar streams reales, reemplaza el contenido de player-embed con un <iframe> apuntando a la URL del video:
  • Usa tu propio backend para obtener URLs desde AnimeFLV API, Gogoanime o HiAnime
  • Inserta el iframe: <iframe src="URL_STREAM" allowfullscreen></iframe>
  • Servidores soportados: StreamSB, Filemoon, Mp4upload, Voe, Doodstream
  • Formato recomendado: HLS (.m3u8) con hls.js para reproductor nativo
Ejemplo de iframe real:
<iframe src="https://www.gcdneu.pw/embed/CODIGO" allowfullscreen scrolling="no" allow="autoplay"></iframe>
